# Data Directory

This directory contains sample data streams and historical data for the Bharat-Grid AI system.

## Structure

- `nodes_stream.csv` - Simulated energy node data stream
- `supply_stream.jsonl` - Simulated supply event data stream  
- `historical/` - Historical consumption patterns for RAG training

## Data Generation

Sample data generators will be implemented in Task 10.1 to create realistic energy node data streams, supply event simulation data, and historical consumption patterns for RAG training.

## Usage

The Pathway framework will read from these data sources to simulate real-time energy distribution scenarios during development and testing.
